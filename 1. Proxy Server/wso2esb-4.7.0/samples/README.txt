This directory contains Axis2 based client/server samples.

== axis2Client == 
The axis2Client folder contains an Apache Ant based project that contains clients for the different sample services provided in axis2Server. 

== axis2Server == 
This directory contains several Apache Ant based backend services. Compiled services can then be deployed by running the axis2server scripts. 


For more information on the WSO2ESB samples, please go to http://docs.wso2.org/
